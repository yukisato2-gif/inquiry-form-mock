# 現場の声 — モック環境

社内向け「現場の声」受付システムのモック環境です。
本リポジトリはモックであり、**本番環境ではありません**。
要件確認・画面確認・将来のシステム化前段の検討用として使用します。

## 公開URL（モック）

- 投稿フォーム: https://inquiry-form-mock.vercel.app/post
- 公開は Vercel 上のモック環境です。

## 主な用途

- 画面・導線の要件確認
- 関係者とのイメージすり合わせ
- 今後の正式システム化に向けた土台

## 確認方法

| 環境 | 用途 |
|------|------|
| **Vercel URL** | 正式な確認先。修正後はここで確認してください |
| `localhost:3000` | 開発中の一時確認用 |

## データについて

- ブラウザの localStorage にデータを保存しています
- 初期サンプルデータは 18 件です
- データが表示されない場合は、ダッシュボード下部の「サンプルデータに戻す」で復元できます

## ローカル起動

```bash
npm install
npm run dev
```

`start-dev.bat` をダブルクリックでも起動できます（Windows）。

## 技術構成

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Zustand (状態管理 + localStorage 永続化)
- Recharts (グラフ)

## 注意事項

- 本リポジトリはモックです。**本番環境では使用しないでください**。
- 本番データは一切使用していません。表示しているデータはすべてモック用のサンプルです。
- データ保存はブラウザの localStorage のみで、サーバ側への保存は行っていません。
- 検証・要件確認以外の目的での使用は想定していません。
